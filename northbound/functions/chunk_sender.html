NECTOR Chunk Sender

Creating slices, NC coding at slice-level if required, sending slices over UDP, etc.

  • Chunk sender general
  • Chunk sender idetintied by: IP address, UDP port
  • Chunk sender sub-blocks
  • 1 - Slice creator: create slices from chunks. (1472 bytes)
  • Slice creator output: Source Slice index number, Source Slice size, Source Chunk Payload.
  • 2 - Send/Receive function: slices are sent over the UDP
  • Send/Receive function gets inputs from NECTOR Client on the Peer (Leecher) IP address and Peer (Leecher) UDP/TCP port numbers
  • 3 - Grouping Slices into Generations (Optional): The slices are grouped in generations. The generation size is configurable by the user. The user may configure generation size to 8, 16, 32 and 64.
  • 4 - NC Encoder (Optional): The same NC library is used as it was in the case of NC at chunk-level. The code-rate is configured. The output of NC Encoder are coded slices.
